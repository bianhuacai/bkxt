# bkxtpdcdx

本科毕业论文选题可行性判断 Skill。

`bkxtpdcdx` 用于判断一个本科毕业论文选题是否可行，并给出修改路径。它不只回答“能不能做”，而是把题目放到“母题、候选题、可开题题目”三个层级里判断，再从数据、变量、文献、同质化、实证难度等维度给出具体修改建议。

## 适合场景

- 判断本科毕业论文选题是否可行
- 判断一个题目是不是太宽、太空、太难或太同质化
- 分析实证论文的数据和变量能不能落地
- 把一个大方向改成可开题的论文题目
- 为经管、管理、金融、产业、乡村振兴、企业研究等方向做选题初筛

## 评价维度

Skill 会从 9 个维度诊断选题：

1. 母题可行性
2. 题目成熟度
3. 数据可得性
4. 变量可测量性
5. 文献基础
6. 同质化风险
7. 实证难易程度
8. 论文结构可写性
9. 修改弹性

最终会给出四类判断：

- 可直接开题
- 可做但需修改
- 风险较高
- 不建议做

## 安装

下载本仓库后，将 `bkxtpdcdx` 文件夹复制到你的 Skills 目录。

Windows 示例：

```text
C:\Users\你的用户名\.agents\skills\bkxtpdcdx
```

macOS / Linux 示例：

```text
~/.agents/skills/bkxtpdcdx
```

复制后，重新打开 Codex 或新建对话，即可调用。

## 使用

在对话中输入：

```text
$bkxtpdcdx 企业数字化转型对绿色创新
```

也可以这样使用：

```text
$bkxtpdcdx ESG 对汽车行业出口绩效
```

```text
$bkxtpdcdx 宜宾茶旅融合政策对农村居民收入
```

## 输出内容

典型输出包括：

- 当前判断：可直接开题 / 可做但需修改 / 风险较高 / 不建议做
- 当前层级：母题 / 候选题 / 可开题题目
- 一句话结论
- 9 个维度评分
- 主要风险
- 修改建议
- 可改写的论文题目

## 设计原则

这个 Skill 的核心判断不是“选题高级不高级”，而是：

> 这个本科生能不能在现实约束下把它写完，并写成一篇合格的毕业论文。

因此，它会优先考虑数据可得性、变量可测量性、文献支撑、实证难度和修改空间。
